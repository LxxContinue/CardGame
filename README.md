福建十三水的游戏，从服务器获取牌面，并作出最优解

- 运行环境：该项目运行于Xcode 10+ 不支持使用lbstdc++插件报错异常解决方案
类似这种报错Info: `Error: ld: library not found for "-lstdc++.6"`

- 编译方法：xcode编译器编译

- 使用方法：clone项目后可在xcode上开模拟器运行。如果无此编译器看博客里的视频即UI演示
